# `@itsrakesh/ui`

UI components for itsrakesh projects.

## Installation

```bash
npm i @itsrakesh/ui

or

yarn add @itsrakesh/ui

or

pnpm add @itsrakesh/ui
```

## Usage

```tsx
import { Button } from "@itsrakesh/ui";

export default function App() {
  return (
    <div>
      <Button>Click Me</Button>
    </div>
  );
}
```
